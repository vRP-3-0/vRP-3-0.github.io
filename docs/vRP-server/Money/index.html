Money

Get Cash​

Gets the current cash on the user

local cash = vRP.getMoney(user_id)
print(cash) -- returns number

Set Cash​

Overrules the current cash and assignes a new value

vRP.setMoney(user_id, 6000) -- new value is 6000 in cash

Try payment​

Try to take a certain amount from the player

if vRP.tryPayment(user_id, 6500) then
    print("payed for the thing")
end

Give Money​

Give money to the specified user_id

vRP.giveMoney(user_id, 7000)

Get Bank Money​

Gets the current bank money the user has

local bankMoney = vRP.getBankMoney(user_id)
print(bankMoney) -- returns number

Set Bank Money​

Overrules the current bank money and assignes a new value

vRP.setBankMoney(user_id, 6000) -- new value is 6000 in the bank

Give Bank Money​

Give bank money to the specified user_id

vRP.giveBankMoney(user_id, 6500)

Try to withdraw money​

Try to take money from the bank

if vRP.tryWithdraw(user_id, 6500) then
    print("got the money in cash")
end

Try to deposit money​

Try to depost money into the bank

if vRP.tryDeposit(user_id, 6500) then
    print("money got deposited")
end

Try a full payment wallet + bank​

If you don't have enough in the wallet it will take from your bank account

if vRP.tryFullPayment(user_id, 2500) then
    print("payed all")
end
